Projet DevLab

Le Devlab est un projet de création de site e-commerce dans le but de commercialiser les goodies des associations du pôle Leonard de Vinci. Ce projet est encore en cours. A ce jour, nous avons pu intégrer les fonctionnalités principales de notre maquette en utilisant wordpress(.org).

Projet BAP Studio-Edoras

Ce projet avait pour but de réaliser une landing page pour l'entreprise Studio-Edoras. Cette landing page devait permettre aux utilisateurs de pouvoir être en contacter directement par l'entreprise après qu'ils aient entré leurs coordonnées sur le formulaire de la page contact

Projet URC Draveil

Ce projet avait pour but la réalisation d'un événement de promotion pour une association de roller dans la ville de Draveil. Pour ce projet, nous avons créer l'évenement "Roller Day". Ce jour là, nous avons organisé une randonnée de roller dans une grande partie de la ville ainsi qu'une après-midi initiation pour les personnes intéressées par l'association. Nous avons effectué plusieurs missions comme la conception d'affiche, la communication par mail et sur les réseaux sociaux, la négociation pour obtenir l'autorisation de la mairie, ...
